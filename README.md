# uPyCraft
uPyCraft is an IDE designed for micropython. This is my PyQt5 port.

to run this software, you need python 3.6, pyqt5 and pyserial.
![](https://github.com/XIVN1987/uPyCraft/blob/master/截图.png)

### file/folder operaton
+ create
+ delete
+ rename
+ drag
+ drag from PC
+ save to PC

### file opertion
+ upload
+ editing
+ download
+ run
